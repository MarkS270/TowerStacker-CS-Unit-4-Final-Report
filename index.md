# Tower Stacker
## Introduction
In this project, I had to create a simple game by first giving a 50-limit prompt to any AI tool to generate this game in Java Swing version. The project tested my understanding of layout managers, user interaction, event handling, file I/O, collision detection, and object-oriented programming through the use abstract classes and inheritance.

## What is does
The game I chose to make was Tower Stacker.
The game challenges users to stack properly as many blocks as possible, trying to get as large of a score as possible. This tower stacker game allows the player to input a username, play tower stacker and record highscore to compare to other players. The code also allow the user to change the difficulty of the game to their liking.
